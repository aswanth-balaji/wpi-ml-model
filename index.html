Key Finding — Volatile Matter Dominates
Volatile matter content (feature importance: 38.9%) is the single biggest lever for combustion efficiency. Switching from PET (VM~60%) to PE/PP (VM~95%) alone adds ~8% CE. Moisture (16.8%) and injection rate (11.2%) follow. This validates JFE Steel's coarse-plastic research and the pre-processing focus in industrial plants.

Industrial Loopholes Identified
1. Injection Rate Cliff: CE peaks at 36–40 kg/tHM but current plants run 50–60. Model identifies 12–18% CE loss in this zone — actionable via O₂ co-injection.
2. PVC Contamination Signature: VM drop below 75% is a strong proxy for chlorine-bearing plastic entering the feed — triggers automatic interlock recommendation.
3. Moisture×VM Interaction: Wet feed (>1.5%) with low VM (<80%) creates compounding CE penalty not captured by linear monitors — novel finding.
4. O₂ Saturation: O₂ enrichment above 3.5% yields diminishing returns. Wasted cost that current plants do not know to avoid.

Novel Contribution vs Current Practice
JSW+JFE pilot (2024) uses rule-based thresholds. This model captures nonlinear interactions — particularly the VM×moisture compound penalty and the O₂ saturation curve — which threshold systems miss. Integration with DCS (Distributed Control System) enables predictive injection control 30–60 seconds ahead of condition change.
Deployment Path
Step 1: Shadow mode — model runs alongside human operator, logs divergence. Step 2: Advisory mode — real-time injection rate recommendation displayed on HMI. Step 3: Closed-loop — automatic injection rate trimming within ±5 kg/tHM of operator setpoint. Each stage requires ISA-88 validation and HAZOP review.
